<h1>Tic-Tac-Toe Game</h1>

<p>A simple Tic-Tac-Toe game built using HTML, CSS, and JavaScript. This project allows two players to play the classic Tic-Tac-Toe game in a browser.</p>

<h2>Features</h2>
- Two-player mode
- Interactive user interface
- Real-time game updates
- Displays winner or draw status
- Responsive design for different screen sizes

<h2>Technologies Used</h2>
- HTML
- CSS
- JavaScript

<h2>How to Run the Project</h2>
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/tic-tac-toe.git
   ```
2. Navigate to the project folder:
   ```bash
   cd tic-tac-toe
   ```
3. Open the `index.html` file in your browser.

<h2>How to Play</h2>
1. The game starts with Player X.
2. Players take turns clicking on empty squares to place their mark (X or O).
3. The first player to get three marks in a row (horizontally, vertically, or diagonally) wins.
4. If all squares are filled without a winner, the game ends in a draw.
5. Click the "Restart" button to play again.
