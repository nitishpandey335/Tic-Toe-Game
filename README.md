    <h1>Tic-Tac-Toe Game</h1>
    <p>A simple Tic-Tac-Toe game built using HTML, CSS, and JavaScript. This project allows two players to play the classic Tic-Tac-Toe game in a browser.</p>
    
    <h2>Features</h2>
    <ul>
        <li>Two-player mode</li>
        <li>Interactive user interface</li>
        <li>Real-time game updates</li>
        <li>Displays winner or draw status</li>
        <li>Responsive design for different screen sizes</li>
    </ul>
    
    <h2>Technologies Used</h2>
    <ul>
        <li>HTML</li>
        <li>CSS</li>
        <li>JavaScript</li>
    </ul>
    
    <h2>How to Run the Project</h2>
    <ol>
        <li>Clone the repository:</li>
        <pre><code>git clone https://github.com/your-username/tic-tac-toe.git</code></pre>
        <li>Navigate to the project folder:</li>
        <pre><code>cd tic-tac-toe</code></pre>
        <li>Open the <code>index.html</code> file in your browser.</li>
    </ol>
    
    <h2>How to Play</h2>
    <ol>
        <li>The game starts with Player X.</li>
        <li>Players take turns clicking on empty squares to place their mark (X or O).</li>
        <li>The first player to get three marks in a row (horizontally, vertically, or diagonally) wins.</li>
        <li>If all squares are filled without a winner, the game ends in a draw.</li>
        <li>Click the "Restart" button to play again.</li>
    </ol>
